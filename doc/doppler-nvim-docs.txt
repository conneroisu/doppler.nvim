FEATURES AND STRUCTURE*doppler-nvim-docs-doppler-neovim-plugin-features-and-structure*

Injects doppler secrets into your neovim environment based on the doppler
configs for the project.


INSTALLATION/USAGE*doppler-nvim-docs-doppler-neovim-plugin-installation/usage*

Lazy.nvim is a plugin manager that allows you to load plugins on demand. To
install this plugin, add the following to your `init.lua`

>lua
    return {
        "conneroisu/doppler.nvim",
        dependencies = {
            'nvim-lua/plenary.nvim',
        },
    }
<


WHY?                            *doppler-nvim-docs-doppler-neovim-plugin-why?*

Sometimesyour tests require environmental secrets.

This plugin allows you to automatically inject them into your neovim
environment when launching neovim inside of a known doppler project.
